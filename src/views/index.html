Overview

This UI toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Getting Started

Start by adding the following files to your page:

<link rel="stylesheet" href="assets/toolkit/styles/toolkit.css">

Storysettings’ JavaScript requires jQuery

<script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script src="assets/toolkit/scripts/toolkit.js"></script>

Using Storysettings

Storysettings is made of up of Components, Layout, Utilities, States, and Base. As you are building your story use those parts to design it with your content.

You can see how you can use Storysettings in this example.

Sass

If you prefer to customize Storysettings you can download this repository and modify /src/assets/toolkit/styles/_config.scss and output a new toolkit.css or install using Bower.
